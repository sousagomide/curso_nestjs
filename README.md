# Curso de NestJS

<h1>Requerimentos</h1>

<ul>
    <li>NodeJS</li>
    <li>NPM</li>
</ul>

<h1>Instalar o NestJS CLI</h1>

<code>
    npm install -g @nestjs/cli
</code>

<br/>Testando a instalação

<code>
    nest --version
</code>

<h1>Criar uma nova aplicação</h1>

<code>
    nest new <i>nome_aplicacao</i> --skip-git
</code>

<br/>Escolha a opção:
<ul>
    <li>npm</li>
</ul>

<h1>Rodar o servidor</h1>

<code>
    npm run start
</code>

<br/>Host: <code>localhost:3000</code>
